# Fuentes corporativas Alcocars

Coloca aquí los archivos `.woff2` de las fuentes propietarias del cliente.

## Archivos requeridos

| Archivo | Fuente | Peso | Uso |
|---|---|---|---|
| `barracuda-script.woff2` | Barracuda Script | normal | Logo / marca |
| `sisco.woff2` | Sisco | 400 | Títulos y display |
| `sisco-bold.woff2` | Sisco | 700 | Títulos negrita |
| `avant-garde.woff2` | Avant Garde | 400 | Cuerpo de texto, UI |
| `avant-garde-bold.woff2` | Avant Garde | 700 | UI negrita |

## Notas

- Estas fuentes son licencias propiedad del cliente.
- Mientras los archivos no estén presentes, el sitio usará `system-ui` como fallback (funcional, diferente visualmente).
- NO subir los archivos al repositorio sin confirmar la licencia de distribución.
